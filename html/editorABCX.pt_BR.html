Editor de Partituras

As três abas de partituras (Músicas, acordes e exercícios), conforme já dissemos, são formadas por peças no formato ABCX (ver documentação). Estas três abas possuem algumas funcionalidades que podem ser acessadas através da "Barra de Botões" que fica localizada à direita das mesmas. As funcionalidades são: imprimir, editar ou executar uma partitura.

Barra de botões

A barra de botões possui dois modos de operação: o modo normal e o modo didático (ver documentação do modo didático). No modo normal, estão disponíveis os 5 botões listados acima:

  • Modo normal/didático: chaveia entre os dois modos.
  • Executar/Pausar: inicia a execução da partitura que estiver visível ou a suspende, caso já esteja em execução.
  • Parar: para a execução atual e posiciona no primeiro compasso.
  • Ferramentas: dá acesso ao editor de partituras (e outras funcionalidades).
  • Imprimir: abre a tela para visualizar e imprimir a partitura ativa.

No modo normal, o executor de MIDI não requer explicações adicionais, visto que só possui as funcionalidades básicas, isto é, executar/pausar/parar. A impressão é realizada com as ferramentas do próprio browser. Já o botão "Ferramentas" dá acesso ao Editor de Partituras, que é o foco desta página.

Ferramentas

Ao clicar no menu ferramentas, uma nova página será exibida. Esta página é dividida em 3 partes: a primeira delas é uma área de texto que expõe o conteúdo do arquivo ABCX. Abaixo desta, fica a "Barra de botões de edição" e, ao final, a área de visualização da partitura resultante do texto.

Barra de botões de edição

Da esquerda para a direita os seguintes botões de funcionalidades são listados:

  • seletor de acordeon: seleciona o acordeon para o qual a tablatura será gerada.
  • seletor de tom: modifica (transpõe) a tonalidade da partitura.
  • oitava acima: transpõe a partitura para uma oitava acima.
  • oitava abaixo: transpõe a partitura para uma oitava abaixo.
  • ver impressão: permite a impressão do estado atual da partitura.
  • salvar local: permite fazer o download do arquivo ABCX do estado atual da partitura.
  • fechar: fecha o editor e retorna a tela de execução, incluindo as alterações feitas na partitura.
  • atualizar: atualiza a partitura a partir das alterações efetuadas no texto ABCX. Após a atualização a lista de erros (se houver) será mostrada juntamente com a partitura.
  • Play/Pause/Stop/Relógio: um tocador de MIDI simples está disponível para a execução da partitura. Para que o MIDI esteja disponível é necessário atualizar a partitura pelo menos uma vez.

O editor é bastante simples mas pressupõe o conhecimento do formato ABC, bem como das regras usadas no formato ABCX para gerar a tablatura. Para aprender o formato ABC procure documentação na internet.

Quanto às funcionalidades que este editor provê, destacamos as seguintes características:

  • Sempre que o texto ABCX for alterado e a chave "accordionTab" estiver presente, se não houver uma tablatura definida, uma nova será criada; se já existir ela ficará desatualizada. Este comportamento evita que se perca edições manuais realizadas na tablatura.
  • Para forçar a atualização da tablatura, apague as linhas após "accordionTab" e clique em "Atualizar": uma nova tablatura será gerada automaticamente.
  • Usar o seletor de tom ou os botões de oitavas irá modificar o arquivo ABCX, mas qualquer tablatura que existir ficará desatualizada, pelos motivos expostos acima.
  • O objetivo de transpor a tonalidade é permitir carregar o repertório atual em um acordeon com afinação diferente e ainda assim produzir a tablatura.
  • Salvar uma partitura não significa que o arquivo original no site foi alterado: apenas se está gerando uma cópia no computador local, permitindo ao usuário ter um repertório próprio, contendo as alterações que foram consideradas convenientes e que podem ser carregadas sempre que o site for acessado, usando o menu "Repertório", opção "Carregar do drive local"
